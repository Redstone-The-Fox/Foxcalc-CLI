# Foxcalc-CLI 
### CLI Version of [Foxcalc](https://github.com/Redstone-The-Fox/Foxcalc)
#### Current version: 1.0-b

Some math functions are supported:
```
+ - * / ^ ( ),
sin(),
sinh(),
cos(),
cosh(),
tan(),
tanh(),
pi,
sqrt(),
ceil(),
floor(),
ans()
```

Docker image: https://hub.docker.com/repository/docker/redstonethefox/foxcalc-cli
