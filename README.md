# Foxcalc-CLI 
### CLI Version of [https://github.com/Redstone-The-Fox/Foxcalc](Foxcalc)

Some math functions are supported:
```
+ - * / ^ ( ),
sin(),
sinh(),
cos(),
cosh(),
tan(),
tanh(),
pi,
sqrt(),
ceil(),
floor(),
ans()
```

Docker image: https://hub.docker.com/repository/docker/redstonethefox/foxcalc-cli
